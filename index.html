Set the radius of the touch probe tip. This will usually be specified in the documentation for the touch probe, and is used to compensate for the size of the touch probe on horizontal probing operations.
Set the deflection distance that the touch probe experiences before activating. This will need to be measured once the machine is running, so it is fine to leave this at 0 for initial configuration. You can measure the dimensions of a known work-piece using G6000, and the difference in the probed dimension vs. the known dimension is twice the deflection value. This may take a couple of tries to get right.
